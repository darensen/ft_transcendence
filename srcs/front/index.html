voici ici le meilleur pong du monde !!

le jeu est en cours de développement

comment jouer ?

utiliser les flèches du clavier pour déplacer la raquette

le but est de marquer le plus de points possible

vous pouvez jouer seul ou à deux

pour jouer à deux, utilisez les touches A et D pour le joueur 1 et les flèches du clavier pour le joueur 2

le jeu se termine lorsque le joueur 1 ou le joueur 2 atteint 10 points

vous pouvez également jouer contre l'ordinateur
